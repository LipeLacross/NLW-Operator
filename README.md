## 📂 Projetos

Este repositório contém dois projetos principais:

### 1. [DevRoast](./nlw-operator-fullstack-devroast-main) — Full-Stack

Analisador de qualidade de código que dá uma nota brutalmente honesta de 0 a 10. Envie qualquer trecho de código, ative o "roast mode" para sarcasmo máximo, e descubra o quão ruim (ou bom) seu código realmente é.

- **Stack**: Next.js 16, tRPC, React Query, Drizzle ORM, PostgreSQL, Tailwind CSS, AI SDK (OpenAI)
- **Funcionalidades**: Submissão de código, Roast mode, Análise detalhada, Sugestões de fix, Shame leaderboard

### 2. [Computer Vision](./nlw-operator-computer-vision-main) — Python

Coleção de projetos e experimentos de Visão Computacional, focando em Deep Learning e Reconhecimento de Gestos em tempo real.

- **Stack**: Python 3.14+, PyTorch, MediaPipe, FastHTML, OpenCV, YOLO, CLIPSeg
- **Módulos**: LeNet-5 MNIST, Recognition System & Lab, Computer Vision App
