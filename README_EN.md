## 📂 Projects

This repository contains two main projects:

### 1. [DevRoast](./nlw-operator-fullstack-devroast-main) — Full-Stack

Code quality analyzer that gives a brutally honest score from 0 to 10. Submit any piece of code, enable "roast mode" for maximum sarcasm, and discover how bad (or good) your code really is.

- **Stack**: Next.js 16, tRPC, React Query, Drizzle ORM, PostgreSQL, Tailwind CSS, AI SDK (OpenAI)
- **Features**: Code submission, Roast mode, Detailed analysis, Fix suggestions, Shame leaderboard

### 2. [Computer Vision](./nlw-operator-computer-vision-main) — Python

Collection of Computer Vision projects and experiments, focusing on Deep Learning and Real-Time Gesture Recognition.

- **Stack**: Python 3.14+, PyTorch, MediaPipe, FastHTML, OpenCV, YOLO, CLIPSeg
- **Modules**: LeNet-5 MNIST, Recognition System & Lab, Computer Vision App
